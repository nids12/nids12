## 🚀 Featured Projects

### 🎯 [SmartEvents](https://github.com/nids12/event-management-platform) — Full-Stack Event Platform
> FastAPI · React · SQLite · JWT · REST APIs

A production-grade event management system with **100+ concurrent user support**, waitlist promotion logic, and an in-app notification system. The interesting engineering problem: making registration atomic under concurrent load so two users can't both claim the last slot.

**What I'm proud of:** The concurrency-safe waitlist promotion — cancellation and waitlist assignment run in the same transaction.

---

### 📈 [FinPulse](https://github.com/nids12/Finpulse) — Real-Time Stock Dashboard
> Node.js · Express · WebSockets · MongoDB · Chart.js

Real-time stock data streaming with **sub-second latency** via WebSockets across **50+ simultaneous connections**. Chose WebSockets over polling deliberately — the difference in UX when watching live prices is immediate.

**What I'm proud of:** Clean connection lifecycle management — dead clients are cleaned up immediately, no memory leaks in the broadcast registry.

---

### 🔍 [DocuChat](https://github.com/nids12/Docuchat) — RAG-Powered Document Query
> FastAPI · Qdrant · SentenceTransformers · Ollama · AWS

A full RAG pipeline from scratch: PDF → chunked with overlap → embedded with `all-MiniLM-L6-v2` → stored in Qdrant → retrieved by semantic similarity → answered by a local LLM. **~40% improvement** in retrieval accuracy over keyword search.

**What I'm proud of:** Understanding why chunk overlap matters and how retrieval quality degrades without it.

---

### 🤝 NGOConnect — JP Morgan Code for Good 2024
> React · Node.js · MongoDB · Twilio · OAuth · JWT

Built in **24 hours** with a team of 4 for a real NGO. Requirement tracking platform with Twilio SMS notifications, OAuth login, and role-based access. The NGO's feedback after the demo: the Twilio alerts were the feature they'd use immediately.

**What I'm proud of:** Shipping something real, for a real organization, under extreme time pressure.
